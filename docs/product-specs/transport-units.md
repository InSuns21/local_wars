# transport-units

輸送車と輸送ヘリ、および輸送コマンドの仕様案です。

最終更新: 2026-03-07
状態: 草案

## 対象スコープ
- 輸送車の追加
- 輸送ヘリの追加
- 搭載と降車コマンドの追加
- キャリー中ユニットの状態管理
- UI と AI に必要な輸送導線

## 依頼条件から確定している仕様

### ユニット種別
- 地上輸送ユニットとして `輸送車` を追加する
- 航空輸送ユニットとして `輸送ヘリ` を追加する
- 輸送車は `工場` で生産する
- 輸送ヘリは `空港` で生産する
- どちらも `攻撃不可` とする
- どちらも `反撃不可` とする

### 防御バランス
- 輸送車の防御力は `偵察車レベル` とする
- 輸送ヘリの防御力は `対空攻撃に対して航空ユニット中で最弱レベル` とする
- 輸送ユニットは前線輸送能力の代償として、直接戦闘には弱い支援ユニットとする

### キャリー容量
- 輸送車は `陸上ユニットを最大 2 ユニット` までキャリーできる
- 輸送ヘリは `歩兵を最大 1 ユニット` のみキャリーできる

### キャリー中の燃料
- 輸送ユニットが移動しても、キャリーされているユニットの燃料は減らない
- キャリー中ユニットの燃料消費停止は、輸送中に限って適用する

### 破壊時の扱い
- 輸送ユニットが破壊されたとき、キャリー中ユニットもまとめて破壊される
- キャリー中ユニットの生存判定や緊急降車は行わない

## 提案している補完仕様

### 搭載対象
- 輸送車の搭載対象は `歩兵 / 偵察車 / 対戦車 / 自走砲 / 対空車 / 補給車 / 高射砲車 / 地対空ミサイル車` とする
- `戦車 / 重戦車 / 海上 / 航空 / 輸送ユニット` は輸送車に搭載不可とする
- 輸送ヘリの搭載対象は `歩兵のみ` とする
- 搭載対象は `味方ユニットのみ` とする
- HP が 0 以下のユニットは搭載不可とする

### 搭載コマンド
- 輸送ユニットは `搭載` コマンドを持つ
- 搭載対象は `マンハッタン距離 1` の味方ユニットとする
- 搭載時、対象ユニットは盤面から一時的に取り除かれ、輸送ユニット内部の `cargo` に入る
- 搭載されたユニットはそのターン中に追加行動しない
- このターンに搭載したユニットは同一ターン中に降車できない
- 搭載実行後も、輸送ユニットは移動可能とする
- 輸送ユニットが移動後でも、隣接条件と容量条件を満たす限り `搭載` を実行可能とする
- `搭載` は 1 ターンに `cargoCapacity` 回までとする

### 降車コマンド
- 輸送ユニットは `降車` コマンドを持つ
- 降車先は輸送ユニットに隣接する空きタイルとする
- 1 回の降車コマンドで 1 ユニットだけ降ろす
- 輸送車は地上ユニットが侵入可能なタイルにのみ降車できる
- 輸送ヘリは歩兵を `海以外` のタイルへ降車できる
- 降車後のユニットは `moved=true, acted=true` とし、そのターンの追加行動は不可とする
- このターンに降車したユニットは同一ターン中に再搭載できない
- 降車実行後も、輸送ユニットは移動可能とする
- 輸送ユニットが移動後でも、降車条件を満たす限り `降車` を実行可能とする
- `降車` は 1 ターンに `cargoCapacity` 回までとする

### 盤面占有ルールとの整合
- 既存どおり `1 マス 1 ユニット` を維持する
- 輸送ユニットと搭載ユニットの同居は、盤面上ではなく `cargo` データで表現する
- 搭載中ユニットは盤面上のユニット数に含めない

### 補給と回復
- 搭載中ユニットは施設補給や補給ユニットの対象外とする
- 搭載中ユニットは HP 回復しない
- 燃料減少だけを停止し、弾薬は現状維持とする
- 輸送ヘリ自身は通常の航空ユニットとして燃料管理を受ける

### 生産とコスト差
- 輸送車は `低〜中コスト` の地上支援ユニットとする
- 輸送ヘリは `中コスト以上` の航空支援ユニットとする
- コストは、輸送車 < 輸送ヘリ を原則とする

## 未確定事項
- AI が輸送ユニットを生産する優先度と、輸送対象ユニットの選定規則

## 推奨方針
- 輸送車の搭載対象には `歩兵 / 偵察車 / 対戦車 / 自走砲 / 対空車 / 補給車 / 高射砲車 / 地対空ミサイル車` を含める
- `戦車 / 重戦車 / 海上 / 航空 / 輸送ユニット` は輸送車に搭載不可とする
- 輸送ヘリは `移動後に降車可` とする
- 輸送ユニットは `移動後に搭載可 / 移動後に降車可` とする
- 降車可否は `地形と占有だけで判定` する
- 降車先が敵 ZOC に隣接していても許可する

## 受け入れ条件
- 輸送車を工場で生産できる
- 輸送ヘリを空港で生産できる
- 輸送ユニットは攻撃も反撃もできない
- 輸送車は地上ユニットを 2 ユニットまで搭載できる
- 輸送ヘリは歩兵を 1 ユニットだけ搭載できる
- 搭載中ユニットは盤面から消え、輸送ユニットの cargo として保持される
- 搭載中ユニットは輸送ユニットの移動で燃料が減らない
- 搭載後も輸送ユニットは移動できる
- 降車後も輸送ユニットは移動できる
- 輸送ユニットは移動後でも搭載、降車できる
- 同一ターンに搭載したユニットは降車できず、同一ターンに降車したユニットは再搭載できない
- 搭載は 1 ターンに `cargoCapacity` 回まで、降車も 1 ターンに `cargoCapacity` 回までである
- 輸送ユニット破壊時、搭載中ユニットも破壊される
- UI 上で搭載数と搭載中ユニットが分かる
- AI とログが輸送行動を破綻なく扱える

## テスト観点
- unit
  - 輸送車は 2 ユニットまで搭載できる
  - 輸送ヘリは歩兵 1 ユニットのみ搭載できる
  - 攻撃不可、反撃不可が固定される
  - 破壊時に cargo も消滅する
  - 輸送中ユニットの燃料が減らない
  - 搭載は 1 ターンに `cargoCapacity` 回、降車も 1 ターンに `cargoCapacity` 回に制限される
- integration
  - 工場で輸送車、空港で輸送ヘリを生産できる
  - 搭載、移動、降車の一連が成功する
  - 輸送ユニット破壊で cargo も消える
- ui
  - 搭載可能時に搭載コマンドが出る
  - 搭載数が表示される
  - 搭載/降車の使用回数が分かる
  - 降車候補が分かる
  - 輸送ログが表示される
