# suicide-drones-and-counter-drone

自爆ドローン、対ドローン防空車、防空歩兵の仕様案です。

最終更新: 2026-03-08
状態: 草案

## 対象スコープ
- 自爆ドローンの追加
- 対ドローン防空車の追加
- 防空歩兵の追加
- ドローン専用の生産例外ルール
- 対ドローン迎撃ルール
- ドローン戦を活かすマップ追加
- 設定プリセットと個別設定への追加

## 依頼条件から確定している仕様

### 自爆ドローン
- `自爆ドローン` をユニットとして追加する
- 自爆ドローンは `工場` からのみ生産できる
- 自爆ドローンは `DRONE` サブカテゴリを持つ航空系ユニットとして扱い、地形防御補正を受けない
- 自爆ドローンのコストは `1500` とする
- 索敵半径は `3` とする
- 移動力は `6` とする
- 自爆ドローンは `残弾 1` で運用し、攻撃または反撃に参加した後は消滅する
- 攻撃性能の目安は以下とする
  - 対歩兵: `50`
  - 対戦車系: `35` 前後
  - 対航空機
    - 攻撃機 / 輸送ヘリ / 空中補給機: `50`
    - 爆撃機: `20`
    - 戦闘機: `0`
- 自爆ドローンの生産可否は `設定で切替可能` とし、デフォルトは `OFF` とする
- 設定プリセットからもすぐ切り替えられるようにする

### 工場からの複数生産
- ドローンに限り、`1ターンに同一工場から複数回生産可能` とする
- 配置先は `工場の直上 + 周囲1マス` の合計 5 マスから自動選択する
- 上記 5 マスがすべて埋まっている場合、資金が足りていても生産不可とする
- `1つの工場から生産できるアクティブなドローン数は 5 個まで` とする
- 生産済みドローンが消滅した場合、その工場のカウントは減る

### 対ドローン防空車
- `対ドローン防空車` を追加する
- 移動タイプは `キャタピラ` とする
- コストは `7000` 前後とする
- 通常攻撃は `1-2 マス` の範囲攻撃とする
- 攻撃性能の目安は以下とする
  - 対ドローン: `120`
  - 対航空機: ふつう
    - 戦闘機: `25` 基準
    - 爆撃機: `25` 基準
  - 対地: `歩兵並`
- 特殊能力として `迎撃` を持つ
- 迎撃仕様は以下とする
  - 半径 `2` マスで敵ドローンを迎撃対象にできる
  - 迎撃対象は `ドローンのみ`
  - 迎撃には `弾数消費あり`
  - 同一ターンに迎撃できる回数は `2` 回をデフォルトとし、設定で変更可能にする
  - 迎撃確率は `70%` をデフォルトとし、設定で変更可能にする
- このユニットの生産可否は `自爆ドローンの生産可否に準ずる`

### 防空歩兵
- `防空歩兵` を追加する
- 対航空機は `ふつう` とする
  - 戦闘機に対して `20-30` を目安
  - 対空戦車よりは効かないバランスとする
- 対ドローンは `80` とする
- 対地は `歩兵並` とする
- 移動力は `歩兵より鈍足` とする
- コストは `歩兵の4倍` とする
- このユニットは `設定に関係なく生産可能` とする
- ただし AI は、`ドローンが無効` かつ `空港もないマップ` では防空歩兵を生産しない

### マップ追加
- `ドローンが活躍する見込みのマップ` を複数追加する
- 新マップでは、工場前面の展開余地、迎撃網の抜け道、対ドローン防空車の射界を比較できる配置にする

## 提案している補完仕様

### 自爆ドローンの戦闘ルール
- 自爆ドローンは `通常攻撃射程 1` から始める
- 自爆ドローンは `爆撃コマンドを持たない`
- 自爆ドローンは `占領できない`
- 自爆ドローンは `補給車の対象外` とし、空中補給機による補給対象にも含めない
- 自爆ドローンの `最大燃料は 50` とする
- 自爆ドローンの `ターン終了燃料消費量は 1` とする
- 自爆ドローンは既存航空ユニットと同じ燃料管理を受ける
- 自爆ドローンは `海上ユニットも攻撃対象に含む`
- 自爆ドローンは攻撃が解決した時点で盤面から除去する
- 反撃に参加した場合も、その戦闘解決後に盤面から除去する
- 戦闘解決順は `与ダメージ適用 -> 被ダメージ適用 -> 自爆ドローン除去` で固定する

### ドローン生産の自動配置順
- 自動配置順は以下の固定順を推奨する
  1. 工場タイル直上
  2. 上
  3. 右
  4. 下
  5. 左
- この順は UI とログに明記し、再現性を持たせる

### 工場ごとのドローン上限管理
- ドローンは `生産元工場ID` または `originFactoryCoord` を保持する
- ドローンが移動して工場から離れても、上限 5 のカウントは元工場に紐づく
- 工場が占領されても、既に存在するドローンのカウントは生産元に紐づいたまま減少管理する

### 工場の通常生産との競合
- ドローンだけが複数生産例外を持つ
- 非ドローン生産は従来どおり `1工場1ターン1回` とする
- `そのターンに通常ユニットを生産した工場ではドローン生産不可` とする
- `そのターンにドローンを1回でも生産した工場では通常ユニット生産不可` とする
- 理由は、複数生産例外の境界を明確にして UI と AI を単純化するため

### 対ドローン迎撃ルールの補完
- 迎撃は `敵ターン中` に発生する受動処理として扱う
- 敵ドローンが `移動で半径2内に入った時点` でのみ迎撃判定する
- 迎撃失敗でも `迎撃回数` と `弾数` は消費する
- 複数の対ドローン防空車が射程内にいる場合、`目標に最も近いもの` から順に判定する
- 1回の敵行動に対して複数迎撃はありうるが、ドローンが撃墜された時点で後続迎撃は行わない
- 迎撃回数は `防空車の所有者のターン開始時` に全回復する

### 防空歩兵の役割
- 防空歩兵は `前線随伴できる廉価な対ドローン保険` として置く
- 対ドローン防空車より安く、遅く、迎撃能力は持たない
- ドローンが無効な対局でも完全な死にユニットにならないよう、対地は歩兵並を維持する

### 設定追加案
- `enableSuicideDrones`: 自爆ドローンと対ドローン防空車の有効化。デフォルト `false`
- `droneInterceptionChancePercent`: 迎撃確率。デフォルト `70`
- `droneInterceptionMaxPerTurn`: 対ドローン防空車の1ターン迎撃回数。デフォルト `2`
- `droneAiProductionRatioLimitPercent`: ドローンが有効な時の AI 生産比率上限。デフォルト `50`
- プリセットには少なくとも以下を追加する
  - `標準`: ドローン `OFF`
  - `ドローン戦`: ドローン `ON`, 迎撃 `70% / 2回`, AI 上限 `50%`

## 固定した設計判断
- 自爆ドローンは `工場生産の航空系ユニット例外` ではなく、`DRONE` サブカテゴリを持つ方針とする
- ドローンの複数生産 UI は、工場選択時に `ドローン専用の連続生産ボタン` と `現在数 n/5` を表示する
- `攻撃後消滅` と `反撃後消滅` の戦闘解決順は `与ダメージ適用 -> 被ダメージ適用 -> 自爆ドローン除去` で固定する
- 経過ログに `迎撃成功 / 迎撃失敗 / 残迎撃回数` を出す
- AI は最初は `高価値目標がいる時だけドローン生産` と `工場前の空きスロットが2以上ある時だけ追加生産` に絞る

## 確定した数値
- 自爆ドローンの `最大燃料` は `50`
- 自爆ドローンの `ターン終了燃料消費量` は `1`
- 自爆ドローンは `海上ユニット` を攻撃対象に含む
- 自爆ドローンは `空中補給機で補給不可` とする
- 対ドローン防空車の `最大弾数` は `6`
- 防空歩兵の `moveRange` は `2`
- 防空歩兵の `コスト` は `4000`
- ドローンが有効な時の AI 生産比率上限は `50%` とし、詳細設定で変更可能にする
- ドローン向け新マップは `3件` 追加する

## 受け入れ条件
- 設定 ON のときだけ自爆ドローンと対ドローン防空車を生産できる
- 自爆ドローンは工場から複数生産できる
- 自爆ドローンは工場の直上と周囲1マスに自動配置される
- 生産元工場ごとのドローン上限 5 が守られる
- 自爆ドローンは攻撃または反撃後に消滅する
- 対ドローン防空車は範囲 1-2 の通常攻撃と、半径2の迎撃を持つ
- 迎撃回数と迎撃確率は設定で変更できる
- ドローン AI 生産比率上限 `50%` を詳細設定で変更できる
- 防空歩兵は `moveRange 2 / cost 4000` で常時生産可能とし、AI は不要対局では生産しない
- ドローン戦向けマップが `3件` 追加される

## テスト観点
- unit
  - 自爆ドローンのダメージ表と攻撃後消滅
  - 対ドローン防空車の対ドローン火力、迎撃回数、迎撃確率分岐
  - 防空歩兵の対空バランス
- integration
  - 工場からの複数ドローン生産
  - 自動配置順と 5 スロット満杯時の生産失敗
  - 生産元工場ごとの上限管理
  - 迎撃でドローンが移動中または攻撃前に落ちるケース
- ui
  - 設定 ON/OFF で生産候補が切り替わる
  - 工場UIに `現在ドローン数 n/5` が出る
  - 迎撃ログと残回数が表示される
  - プリセットからドローン戦を選べる
