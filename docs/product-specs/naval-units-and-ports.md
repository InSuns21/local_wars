# naval-units-and-ports

海上ユニットと港湾タイルの正式仕様案です。既に `UnitType` と基礎パラメータは定義済みのため、この文書では「どう動くか」「何を提供するか」を固めます。

最終更新: 2026-03-10
状態: 草案

## 対象スコープ
- `SEA` / `PORT` タイルのゲームルール
- 海上ユニット
  - 駆逐艦
  - 揚陸艦
  - 空母
  - 潜水艦
  - 戦艦
  - 補給艦
- 港湾での生産、補給、修理、占領
- 海上輸送、艦載機運用、潜水艦ステルス
- 海戦を主題にしたスカーミッシュマップ 4 種追加

## 実装前提
- 既存の `PORT` は占領可能、爆撃可能、収入対象の施設タイルとして扱う
- 既存の `SEA` は通行不能地形ではなく、海上ユニット専用の移動地形として扱う
- 既存の `1 マス 1 ユニット` ルールは維持する
- 海上ユニットの同居は行わず、輸送・艦載は `cargo` データで表現する
- 航空ユニットと潜水艦のステルスは既存の「敵施設直上または敵ユニット隣接で可視」を基本ルールとして再利用する

## 確定仕様

### SEA タイル
- `SEA` は海上ユニットと航空ユニットのみ進入可能
- 地上ユニットは `SEA` に進入不可
- `SEA` は収入、修理、補給、生産を提供しない
- `SEA` にいるユニットは地形防御補正を受けない
- `SEA` は占領不可、爆撃による施設破壊対象外とする

### COAST タイル
- `COAST` は旧 `SHALLOW` を改名した地形とする
- `COAST` は陸地扱いとし、地上ユニットと航空ユニットが進入可能
- `COAST` は揚陸艦の降車先として使う
- `COAST` は収入、修理、補給、生産を提供しない
- `COAST` は占領不可、爆撃による施設破壊対象外とする
- `COAST` は地形防御補正を持つが、防御側不利の補正にする
- 海上ユニットは `COAST` に進入できない

### PORT タイル
- `PORT` は `CITY / FACTORY / AIRPORT` と同じく占領可能施設タイルとする
- `PORT` は海ユニットが留まれる海タイルでもあり、同時に陸地としても扱う
- `PORT` の占領耐久と施設 HP は 20
- `PORT` は既存の収入設定 `incomePort` を参照して収入を発生させる
- `PORT` は海上ユニットの生産拠点とする
- `PORT` は海上ユニットの HP 回復、燃料回復、弾薬回復を提供する
- `PORT` の補給と修理は `自軍所有かつ operational=true` のときだけ有効
- `PORT` は爆撃で破壊可能で、破壊時は `operational=false` になり収入、生産、補給を停止する
- `PORT` は歩兵系占領ユニットで占領できる
  - 現行仕様では `canCapture=true` のユニット、つまり歩兵と防空歩兵が対象

### 海上ユニット共通
- 海上ユニットは `NAVAL` 移動タイプを使う
- 海上ユニットは `PORT` でのみ生産できる
- 海上ユニットは `PORT` または `補給艦` でのみ燃料・弾薬を回復できる
- 海上ユニットの HP 回復は `PORT` のみ。補給艦は HP 回復を行わない
- 海上ユニットは `SEA` と `PORT` にのみ進入できる
- `RIVER` や `BRIDGE` は海上ユニットの移動対象に含めない
- 海上ユニットは施設占領不可
- 海上ユニットは地形防御補正を受けない
- 海上ユニットは原則として待機時燃料消費を持たない。燃料は移動によってのみ減る
- ただし潜水艦だけは例外で、毎ターン終了時に燃料を `2` 消費する
- 潜水艦はターン終了時の燃料が `0` になった場合に沈没する
- 潜水艦以外の海上ユニットは燃料 `0` でも沈没しない

## ユニット別仕様

### 駆逐艦
- 役割は前衛、対艦、対潜、対輸送の汎用護衛艦
- `SEA / PORT` を移動する直接戦闘ユニット
- 敵潜水艦への主要カウンターとする
- 敵潜水艦が駆逐艦に隣接した時、その潜水艦は可視になる
- 駆逐艦は空母や戦艦より安価で、数を揃えやすい主力艦とする
- 対地火力は限定的で、沿岸の軽目標支援までに留める

### 揚陸艦
- 役割は地上ユニット輸送と沿岸上陸
- `LANDER` の `cargoUnitTypes` 定義どおり、陸上ユニット全般と自爆ドローンを最大 2 つまで搭載できる
- 搭載中ユニットは盤面から消え、燃料を消費しない
- 揚陸艦が破壊された場合、cargo もまとめて破壊される
- 揚陸艦の搭載と降車は、既存の輸送ユニット仕様に合わせて `1 ターンに搭載 1 回、降車 1 回` とする
- 揚陸艦は `SEA / PORT` を移動し、降車先は原則として隣接する `COAST` または `PORT` のみとする
- ただし揚陸艦自身が `PORT` 上にいる場合は制限を緩和し、搭載中ユニットが通常進入可能で、かつ空いている隣接タイルへ降車できる
- `COAST` は独立タイルとして実装する
- 上陸を想定するマップは、海岸線に `COAST` または `PORT` の上陸帯を明示配置する
- 揚陸艦自身は攻撃能力を持たず、輸送専用艦とする

### 空母
- 役割は航空戦力の前進基地
- 空母は航空ユニットを最大 2 機まで搭載できる
- 搭載対象は定義どおり `FIGHTER / BOMBER / ATTACKER / STEALTH_BOMBER / AIR_TANKER / TRANSPORT_HELI`
- 自爆ドローンは空母搭載対象に含めない
- 空母は `搭載 / 発艦 / 収容` の 3 コマンドを持つ
- 搭載は `PORT` 上または `PORT` に隣接した空母でのみ可能とする
- 発艦は、空母に隣接する空きタイルへ航空ユニットを出すコマンドとする
- 収容は、空母に隣接する味方航空ユニットを cargo に戻すコマンドとする
- 発艦と収容は、既存の cargo ルールと同様に `移動後でも実行可` とする
- 発艦後の航空ユニットは `moved=true, acted=true` とし、そのターンは追加行動不可とする
- 収容された航空ユニットは cargo 中は燃料消費を停止する
- 収容された航空ユニットは毎ターン終了時に、最大燃料と最大弾薬の `50%` ずつ回復する
- `50%` 回復は最小 `1` 保証で丸める
- 艦載中の燃料・弾薬回復量は設定可能とする
- 収容された航空ユニットは毎ターン終了時に HP を `1` 回復する
- 空母が `PORT` に停泊している場合、艦載中航空ユニットの HP 回復量は設定値を使う
- 艦載中航空ユニットの HP 回復量は `通常時` と `停泊時` をそれぞれ設定可能とする
- 停泊時 HP 回復量のデフォルト値は `1` とする
- 空母は弱い対艦・対空攻撃を持つが、主役は艦載機運用である
- 空母 cargo 中の航空ユニットは `AIR_TANKER` の補給対象外とする

### 潜水艦
- 潜水艦は `NAVAL` かつ `isStealth=true` の海上ステルスユニットとする
- FoW の有無にかかわらず、敵からは通常不可視
- 敵ユニットに隣接、または敵所有 `PORT` 直上にいる場合だけ可視になる
- 潜水艦は `SEA` でのみ待ち伏せ価値を持ち、`PORT` 上では可視になりやすい弱点を持つ
- 潜水艦は対戦艦、対空母、対補給艦に強い
- 潜水艦は待機時燃料消費 `2` を持ち、補給線から切り離されると燃料切れで沈没する
- 潜水艦は駆逐艦に明確に弱い
- 潜水艦は航空ユニットを攻撃できない
- 潜水艦は沿岸地上目標も攻撃しない

### 戦艦
- 役割は長射程の艦砲支援と対艦火力
- 戦艦は `1-5` の長射程を持つ
- 戦艦は地上ユニットに高火力を持つ
- 戦艦は `CITY / FACTORY / AIRPORT / PORT` を砲撃できる
- 戦艦の施設攻撃は航空爆撃と同じく `operational=false` 化を発生させる
- 戦艦は対空能力を持たない
- 戦艦は高コスト、高耐久で、駆逐艦より鈍重とする
- 戦艦は潜水艦に弱い

### 補給艦
- 役割は海上補給線の維持
- 補給艦は攻撃不可、反撃不可
- 補給艦は `補給` コマンドを持つ
- 補給対象は `マンハッタン距離 1` の味方海上ユニットのみ
- 補給艦の 1 回の補給で、範囲内の対象すべてをまとめて燃料・弾薬最大まで回復する
- 補給艦は HP 回復を行わない
- 補給艦の補給回数は `maxSupplyCharges` を共通参照する
- 補給艦の補給回数は `自軍 PORT` でターン終了したときに全回復する
- 空母 cargo 中の航空ユニットは補給艦の補給対象外とする

## 既存仕様との接続

### 施設爆撃との接続
- `PORT` は既に航空爆撃の対象施設に含まれているため、海上実装でもそのまま維持する
- 戦艦砲撃でも `PORT` を機能停止にできる
- `HQ` は引き続き砲撃・爆撃で破壊不可

### 輸送ルールとの接続
- 揚陸艦は既存の `cargo` ルールを流用する
- 揚陸艦の搭載中ユニットは燃料消費停止、破壊時同時消滅を輸送車と同じ原則で扱う
- 空母の艦載機も同じ `cargo` データで表現する
- ただし空母は `発艦 / 収容` を持つため、地上輸送とは別 UI を用意する
- 揚陸艦の降車先制約は原則 `COAST / PORT` 固定とする
- ただし揚陸艦が `PORT` 上にいる場合だけ、搭載中ユニットの通常進入ルールで隣接タイルへ降ろせる

### 補給ルールとの接続
- `PORT` は海上ユニットの常設補給拠点とする
- `補給艦` は `補給車 / 空中補給機` の海上版として実装する
- `PORT` 停泊時の HP 回復量は `Factory` と同じ値を使う
  - 既存設定の増加を避けるため、初期実装では `hpRecoveryFactory` を再利用する
- 海上専用の HP 回復設定値は新設しない
- もし空港の HP 回復設定を将来分離する場合のみ、海上も同じ粒度で再検討する

## UI 仕様
- `PORT` 選択時は `海上ユニット生産UI` を表示する
- `PORT` のツールチップには `収入 / 稼働状態 / 補給可否 / 生産可否` を表示する
- 空母選択時は `搭載数`, `艦載中ユニット`, `発艦`, `収容` を表示する
- 揚陸艦選択時は `搭載数`, `搭載中ユニット`, `搭載`, `降車` を表示する
- 潜水艦は不可視中、敵盤面ではアイコンと HP を表示しない
- 潜水艦が戦闘に入った場合の戦闘予測 UI は、通常ユニットと同様に表示してよい
- 戦艦選択時は `砲撃可能範囲` を盤面表示する
- 補給艦選択時は `補給範囲` を盤面表示する

## AI 方針
- `SEA / PORT` が存在しないマップでは海上ユニットを生産しない
- `PORT` があるマップでは、敵海上ユニットや沿岸拠点がある場合にだけ海上ユニットを検討する
- 駆逐艦は敵潜水艦または揚陸艦が見えたときの優先カウンターとする
- 揚陸艦は、陸路が遠く中央拠点へ最短 4 ターン以上かかる島マップで優先度を上げる
- 空母は空港不足のマップでのみ優先候補にする
- 戦艦は `SEA + 陸地施設` の混成マップで、沿岸砲撃が有効な場合に限定する
- 補給艦は味方海上主力が 2 艦以上ある場合にだけ検討する

## 海戦マップ制約
- 初期配置の海上ユニットは `SEA` または `PORT` に置く
- `PORT` は少なくとも 1 つの `SEA` と 1 つの陸地に隣接させる
- `COAST` は少なくとも 1 つの `SEA` と 1 つの `COAST` ではない陸地に隣接させる
- `COAST` は `COAST` 隣接数 2 以下、`SEA` 隣接数 2 以下に制限する
- `CITY / FACTORY / AIRPORT / HQ` は上下左右 4 マスすべてが `SEA` になる配置を禁止する

## 受け入れ条件
- `SEA` は海上ユニット専用移動地形として機能する
- `PORT` で海上ユニットを生産できる
- `PORT` で海上ユニットの修理、燃料回復、弾薬回復ができる
- 駆逐艦、揚陸艦、空母、潜水艦、戦艦、補給艦がルールどおり振る舞う
- 潜水艦は通常不可視で、隣接または敵港直上の時だけ可視になる
- 揚陸艦は cargo を 2 つまで扱え、原則は隣接する `COAST / PORT` に降車できる
- 揚陸艦が `PORT` 上にいる場合のみ、搭載中ユニットの通常進入可否に従って隣接タイルへ降車できる
- 空母は 2 機まで艦載でき、発艦と収容ができる
- 空母は艦載機の燃料・弾薬・HP をターン終了時に回復できる
- 戦艦は長射程砲撃を行える
- 補給艦は海上ユニットの燃料・弾薬補給を行える
- 海戦向けスカーミッシュマップ 4 種が追加される

## 追加予定マップ
- `海 + 島`
  - 島嶼上陸と揚陸艦の価値を学ぶ入門マップ
  - `COAST` と `PORT` による上陸帯を明示して、降車ルールが自然に理解できるようにする
- `アイランドホッピング`
  - 小島を順に中継しながら前進する、揚陸艦と港湾確保が主役のマップ
  - 中間島の `PORT` と `COAST` を奪い合い、補給線を段階的に伸ばす構成にする
- `海 + 大陸`
  - 海上補給線と沿岸砲撃を主題にした中規模マップ
  - 大陸沿岸に複数の `COAST` を置き、上陸地点の選択肢を作る
- `海 + 陸 + 空`
  - 空母、空港、港湾が競合する複合戦マップ
  - 空母艦載回復を活かせるよう、前線港と後方港の価値差を作る
- `空母機動戦`
  - 空港より空母の前進運用が鍵になる海空主題マップ
  - 中央海域の制空権と前線港の確保で、発艦 / 収容 / 艦載回復の使い分けを学べる構成にする
- `海 + 陸 + 空 (ドローン前提)`
  - 港湾、空港、前線工場が干渉する高難度マップ
  - `COAST` による沿岸侵攻と、港湾・空港・工場の三系統運用が干渉する構成にする
