# support-units-and-resupply

補給車と空中補給機、および補給コマンドの仕様です。

最終更新: 2026-03-07
状態: 確定仕様

## 対象スコープ
- 補給車の追加
- 空中補給機の追加
- 補給コマンドの追加
- 補給対象判定
- UI と AI に必要な補給導線

## 確定仕様

### ユニット種別
- 地上補給ユニットとして `補給車` を追加する
- 航空補給ユニットとして `空中補給機` を追加する
- どちらも戦闘ユニットではなく、攻撃コマンドを持たない
- どちらも占領不可とする

### 補給コマンド
- 補給ユニットは `補給` コマンドを持つ
- `補給` は行動コマンドであり、実行後はそのターンの行動を終了する
- 補給コマンドは自ユニットを中心とした `マンハッタン距離 1` の味方ユニットを対象にする
- 同一タイルの自分自身は補給対象に含めない
- 対象ユニットの `燃料` と `弾薬` を最大値まで回復する
- HP は回復しない
- 範囲内に有効な対象が 1 体もいない場合、補給コマンドは実行できない

### 補給対象
- 補給車の補給対象は `地上ユニットのみ` とする
- 空中補給機の補給対象は `航空ユニットのみ` とする
- 海上ユニットはどちらの補給対象にも含めない
- 敵ユニット、中立ユニットは補給対象に含めない

### 行動制約
- 補給ユニットは通常攻撃できない
- 反撃もできない
- 補給コマンド実行ターンは再度補給できない
- 移動後の補給は可能とする
- ただし補給後に追加行動はできない

### 生産と補給線
- 補給車は `工場` で生産する
- 空中補給機は `空港` で生産する
- 補給ユニット自身の燃料・弾薬回復は、既存の施設補給ルールに従う
  - 補給車は地上ユニットとして扱う
  - 空中補給機は航空ユニットとして扱う

### 役割差
- 補給車は前線直後の地上部隊維持を担当する
- 空中補給機は空港から遠い航空部隊の継戦能力維持を担当する
- 空中補給機は補給対象が航空限定である代わりに、高い機動力を前提とする
- 補給車は空中補給機より低コスト、低機動の支援ユニットとする

## 補完方針

### UI
- 補給可能時だけ `補給` コマンドを表示する
- コマンド説明に `周囲 1 マスの味方ユニットへ燃料・弾薬補給` を表示する
- 補給対象ユニットは盤面上でハイライトする
- 実行結果ログには、補給を受けたユニット ID を列挙する

### AI
- 弾薬切れ、燃料不足の味方が近傍にいる場合、補給ユニットは攻勢より補給位置取りを優先する
- 空中補給機は自機の帰投可能性も考慮する
- 補給対象がいない場合は危険地帯へ前進しすぎない

### データモデル
- 補給対象判定は `地上 / 航空 / 海上` の移動種別ベースで判定する
- 補給量は固定値ではなく `maxFuel` / `maxAmmo` までの全回復とする
- 将来の海上補給ユニット追加を考え、補給対象カテゴリは関数化して持つ

## 受け入れ条件
- 補給車は周囲 1 マスの味方地上ユニットだけを補給できる
- 空中補給機は周囲 1 マスの味方航空ユニットだけを補給できる
- 補給で対象ユニットの燃料と弾薬が最大値になる
- 補給ユニットは攻撃も反撃もできない
- 補給対象がいない場合は補給コマンドを実行できない
- UI 上で補給可能範囲と補給結果が分かる

## テスト観点
- unit
  - 補給対象判定が地上 / 航空で正しく分かれる
  - 範囲 1 の味方だけが対象になる
  - 燃料と弾薬が最大値まで回復する
  - HP は回復しない
  - 補給ユニットは攻撃できない
  - 補給対象 0 のときは実行不可になる
- integration
  - 工場で補給車を生産して地上部隊へ補給できる
  - 空港で空中補給機を生産して航空部隊へ補給できる
  - 移動後に補給し、その後は追加行動できない
- ui
  - 補給可能時だけ補給コマンドが出る
  - 対象ユニットがハイライトされる
  - 補給結果がログに表示される
