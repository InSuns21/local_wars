# support-units-and-resupply

補給車と空中補給機、および補給コマンドの仕様です。

最終更新: 2026-03-07
状態: 確定仕様

## 対象スコープ
- 補給車の追加
- 空中補給機の追加
- 補給コマンドの追加
- 補給対象判定
- UI と AI に必要な補給導線

## 確定仕様

### ユニット種別
- 地上補給ユニットとして `補給車` を追加する
- 航空補給ユニットとして `空中補給機` を追加する
- どちらも戦闘ユニットではなく、攻撃コマンドを持たない
- どちらも占領不可とする

### 補給コマンド
- 補給ユニットは `補給` コマンドを持つ
- `補給` は行動コマンドであり、実行後はそのターンの行動を終了する
- 補給コマンドは自ユニットを中心とした `マンハッタン距離 1` の味方ユニットを対象にする
- 同一タイルの自分自身は補給対象に含めない
- 1 回の補給コマンド実行で、範囲内の有効対象をまとめて補給する
- 補給対象ユニットの `燃料` と `弾薬` を最大値まで回復する
- HP は回復しない
- 補給コマンドを 1 回実行するごとに `補給回数` を 1 消費する
- `補給回数` が 0 の場合、補給コマンドは実行できない
- 範囲内に有効な対象が 1 体もいない場合、補給コマンドは実行できない

### 補給対象
- 補給車の補給対象は `地上ユニットのみ` とする
- 空中補給機の補給対象は `航空ユニットのみ` とする
- 海上ユニットはどちらの補給対象にも含めない
- 敵ユニット、中立ユニットは補給対象に含めない

### 行動制約
- 補給ユニットは通常攻撃できない
- 反撃もできない
- 補給コマンド実行ターンは再度補給できない
- 移動後の補給は可能とする
- ただし補給後に追加行動はできない

### 補給回数
- 補給ユニットは `補給回数` を持つ
- `maxSupplyCharges` を設定項目として持たせ、デフォルト値は `4` とする
- 補給ユニットの初期補給回数は `maxSupplyCharges` を使う
- 補給回数は補給コマンドの実行でのみ消費する
- ターン終了時に補給拠点上にいる場合、補給回数は最大値まで全回復する
- 補給車の補給回数回復拠点は `味方の都市・工場・HQ` とする
- 空中補給機の補給回数回復拠点は `味方の空港` とする
- 回数回復判定は燃料・弾薬の施設補給判定と整合するように扱う

### 生産と補給線
- 補給車は `工場` で生産する
- 空中補給機は `空港` で生産する
- 補給ユニット自身の燃料・弾薬回復は、既存の施設補給ルールに従う
  - 補給車は地上ユニットとして扱う
  - 空中補給機は航空ユニットとして扱う
- 補給回数の全回復拠点も上記の施設区分に合わせる

### 役割差
- 補給車は前線直後の地上部隊維持を担当する
- 空中補給機は空港から遠い航空部隊の継戦能力維持を担当する
- 空中補給機は補給対象が航空限定である代わりに、高い機動力を前提とする
- 補給車は空中補給機より低コスト、低機動の支援ユニットとする

## 補完方針

### UI
- 補給可能時だけ `補給` コマンドを表示する
- コマンド説明に `周囲 1 マスの味方ユニットへ燃料・弾薬補給` を表示する
- 補給対象ユニットは盤面上でハイライトする
- 補給ユニット情報に `残補給回数 / 最大補給回数` を表示する
- 実行結果ログには、補給を受けたユニット ID を列挙する
- 補給回数 0 のときはコマンドを disabled 表示し、理由を読めるようにする

### AI
- 弾薬切れ、燃料不足の味方が近傍にいる場合、補給ユニットは攻勢より補給位置取りを優先する
- 空中補給機は自機の帰投可能性も考慮する
- 補給対象がいない場合は危険地帯へ前進しすぎない

### データモデル
- 補給対象判定は `地上 / 航空 / 海上` の移動種別ベースで判定する
- 補給量は固定値ではなく `maxFuel` / `maxAmmo` までの全回復とする
- 将来の海上補給ユニット追加を考え、補給対象カテゴリは関数化して持つ
- 補給ユニットの状態として `supplyCharges` と `maxSupplyCharges` 相当を保持できる形にする
- 設定値 `maxSupplyCharges` を初期化時に各補給ユニットへ反映する

## 受け入れ条件
- 補給車は周囲 1 マスの味方地上ユニットだけを補給できる
- 空中補給機は周囲 1 マスの味方航空ユニットだけを補給できる
- 1 回の補給実行で範囲内の有効対象をまとめて補給できる
- 補給で対象ユニットの燃料と弾薬が最大値になる
- 補給ユニットは攻撃も反撃もできない
- 補給回数はデフォルト `4` で設定変更できる
- 補給回数 0 の場合は補給コマンドを実行できない
- 補給車は味方の都市・工場・HQ、空中補給機は味方の空港でターン終了したときに補給回数が全回復する
- 補給対象がいない場合は補給コマンドを実行できない
- UI 上で補給可能範囲、補給結果、残補給回数が分かる

## テスト観点
- unit
  - 補給対象判定が地上 / 航空で正しく分かれる
  - 範囲 1 の味方だけが対象になる
  - 1 回の補給で範囲内の有効対象がまとめて回復する
  - 燃料と弾薬が最大値まで回復する
  - HP は回復しない
  - 補給ユニットは攻撃できない
  - 補給回数 0 のときは実行不可になる
  - 補給車は都市・工場・HQ、空中補給機は空港でターン終了時に補給回数が全回復する
- integration
  - 工場で補給車を生産して地上部隊へ補給できる
  - 空港で空中補給機を生産して航空部隊へ補給できる
  - 移動後に補給し、その後は追加行動できない
  - 補給回数を消費し、補給拠点でターン終了すると全回復する
- ui
  - 補給可能時だけ補給コマンドが出る
  - 残補給回数が表示される
  - 補給回数 0 のときは補給コマンドが無効になる
  - 対象ユニットがハイライトされる
  - 補給結果がログに表示される
