# design-docs / index

設計ドキュメントの索引です。ここに、正式な設計判断とその参照先を集約します。

## 正式ドキュメント
- [core-beliefs.md](core-beliefs.md): 設計原則と判断基準
- [system-overview.md](system-overview.md): 現行実装基準のシステム設計要約
- [ui-ux-review.md](ui-ux-review.md): UI/UX 判断記録と継続課題

## 使い分け
- 構造、責務境界、将来拡張の前提は `system-overview.md`
- 画面体験や操作性の判断は `ui-ux-review.md`
- より大きな地図としての配置は `../../ARCHITECTURE.md`

## 運用
- 新しい設計文書はここに追記する
- 実装計画そのものは `../exec-plans/` に置く
- 仕様そのものは `../product-specs/` に置く
