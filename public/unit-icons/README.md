# ユニットアイコン

このフォルダには、盤面で使う外部ユニットアイコンを配置できます。
ファイル名はユニット種別と完全一致にしてください。

例:
- `INFANTRY.png`
- `FIGHTER.png`
- `AIR_TANKER.svg`

読み込み順は次のとおりです。
1. `PNG`
2. `SVG`
3. 内蔵アイコンへフォールバック

対応形式:
- `PNG`
- `SVG`

推奨ルール:
- 透過背景
- 単色シルエット推奨
- 正方形キャンバス推奨
- 余白はできるだけ少なく、中央に配置する
- 文字、グラデーション、影は避ける
- 24px から 30px 程度でも判別できる形にする

`SVG` の場合の推奨:
- 白シルエット推奨

`PNG` の場合の推奨:
- 256x256 以上の正方形推奨
- 白または淡い単色シルエット推奨
- アンチエイリアス込みでも外周が崩れないこと

ファイルが存在しない、または読み込みに失敗した場合は、内蔵アイコンを自動で使います。
